# QA Playground

Playwright test suite for [qaplayground.dev](https://qaplayground.dev). Uses Bun and heavily relies on POM and fixtures.

- **Page objects:** `pages/`
- **Tests:** `tests/`

## Running Tests

**Full suite:**
```bash
bun test
```

**Single test (script example):**
```bash
bun run test:nm
```

See all test scripts in `package.json`.
